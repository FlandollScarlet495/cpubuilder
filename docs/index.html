CPUBuilder

PythonでCPUエミュレータを構築するための補助ライブラリ

概要 | API参照 | サンプル

インストール

PyPIからインストールできます（TestPyPIで動作確認済み）。

pip install cpubuilder

TestPyPI から試す場合：

pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple cpubuilder==0.1.0

クイックスタート

まずインポート：

from cpubuilder import BinaryHelper, MemoryHelper, InstructionBase, instruction, DebugTools

簡単なメモリ操作例：

# MemoryHelper の使用例
mem = MemoryHelper(1024)
mem.write(0, 0x12, size=1)
print(mem.read(0))  # 0x12

# BinaryHelper の例
b = BinaryHelper.to_bytes(0x1234, 2)
print(BinaryHelper.from_bytes(b))

命令セットをデコレータで定義：

class MyCPU(InstructionBase):
    @instruction(opcode=0x10, cycles=2, description="Load immediate")
    def load_immediate(self, reg: int, value: int):
        self.registers[reg] = value

詳しい使い方は API参照 と サンプル を参照してください。
