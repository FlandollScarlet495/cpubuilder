examples/simple_cpu.py の実行

パッケージに含めたサンプルは examples/simple_cpu.py にあります。簡易的な命令セットを定義し、LOAD/ADD 系の命令を実行します。

サンプルコード（抜粋）

from cpubuilder import InstructionBase, instruction, MemoryHelper, DebugTools

class SimpleCPU(InstructionBase):
    def __init__(self):
        super().__init__()
        self.registers = [0] * 8
        self.memory = MemoryHelper(65536)
        self.pc = 0
        self.debug = DebugTools()

    @instruction(opcode=0x10, cycles=2)
    def load_immediate(self, reg: int, value: int):
        self.registers[reg] = value
        self.pc += 3

# 実行: python examples/simple_cpu.py

実行方法

  1. リポジトリのルートに移動
  2. 必要ならパッケージをインストール: pip install -e .
  3. サンプルを実行: python examples/simple_cpu.py

他の例や詳細はリポジトリの examples/ ディレクトリを参照してください。
